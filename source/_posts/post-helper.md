# Front-matter 参数
| 参数 | 作用 | 示例 |
| :--- | :--- | :--- |
| **title** | **文章标题** | `title: 深入理解Hexo` |
| **date** | **创建日期** | `date: 2024-01-01 10:00:00` |
| **updated** | **更新日期** | `updated: 2024-01-02 12:00:00` |
| **categories** | **文章分类** | `categories: [教程, Hexo]` |
| **tags** | **文章标签** | `tags: [Fluid, 配置]` |
| **top (或 pinned)** | **首页置顶** | `top: true` 或 `pinned: true`  |
| **except** | **自定义摘要** | `except: 这是文章的简介...`  |
| **math** | **启用LaTeX公式** | `math: true`  |
| **toc** | **显示目录** | `toc: true`  |
| **index_img** | **列表页缩略图** | `index_img: /images/cover.jpg`  |
| **banner_img** | **文章页头图** | `banner_img: /images/banner.jpg`  |
| **comments** | **是否可评论** | `comments: false` |
